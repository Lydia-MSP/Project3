EXTRACT TRANSFORM LOAD

DATABASE

We used a SQLite Database named diabetes.db for this project. The database was created using the sqlite3.exe as follows:

sqlite3 diabetes.db;

There was a table created to hold the dataset. The table was named diabetes_attributes


CREATE TABLE SQL

CREATE TABLE diabetes_attributes(
Pregnancies INTEGER NOT NULL,
Glucose INTEGER NOT NULL,
BloodPressure INTEGER NOT NULL,
SkinThickness INTEGER NOT NULL,
Insulin INTEGER NOT NULL,
BMI INTEGER NOT NULL,
DiabetesPedigreeFunction INTEGER NOT NULL,
Age INTEGER NOT NULL,
Outcome INTEGER NOT NULL
);

EXTRACT(E) and LOAD(L)

Both Extract(E) and Load(L) processes were executed using SQLite's import utility :

.mode csv
.import c:/sqllite/diabetes.csv diabetes_attributes

TRANSFORMATION(T)

The input data elements were direct mapped to the fields in the table without any transformation
